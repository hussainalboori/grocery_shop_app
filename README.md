# 🥑 Grocery Shop App (Day 2 / 10)

Welcome to **Day 2** of my **10 Days, 10 Apps Challenge**! 🚀

This is a beautiful, minimal Grocery Shop App UI built with Flutter. It features a modern design allowing users to browse fresh groceries, add items to their cart, and review their total price in real-time.

## 🌟 Features
- **Clean UI**: A sleek and modern user interface with smooth navigation between the welcome screen, product listing, and shopping cart.
- **Provider State Management**: Seamlessly manages the shopping cart state across different screens using the `provider` package, separating the business logic from the UI.
- **Dynamic Cart**: Easily add items to the cart, remove them, and instantly see the real-time total price calculation updating on the screen.
- **Responsive Layout**: Uses a versatile `GridView` to display grocery items beautifully on screens of different sizes.

## 💡 Inspiration & "The Twist"
This app was heavily inspired by the amazing [Grocery Shop App UI tutorial by Mitch Koko](https://www.youtube.com/watch?v=uSljGJGSl6w).

**My Own Twist:** While following the core tutorial for the aesthetic UI design, I focused heavily on ensuring all the Flutter widgets are perfectly structured. I took extra care diving into state management with `Provider` to make the interaction between the product listing and the cart entirely robust, ensuring no state logic leaks into the presentation layer!

## 🛠️ Built With
- **Flutter & Dart**
- **provider** (for state management)
- **Asset Images** (for beautiful 2D icons)
